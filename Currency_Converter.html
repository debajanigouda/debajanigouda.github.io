Currency Converter (API Based)

Project Overview

Developed a real-time Currency Converter desktop application using Python and Tkinter. The application integrates the ExchangeRate API to fetch live exchange rates and allows users to convert between multiple international currencies dynamically. The project demonstrates API integration, JSON data handling, and event-driven GUI programming.

Key Features

  • Real-time currency conversion
  • Integration with ExchangeRate API
  • Dynamic dropdown currency selection (Combobox)
  • Displays last updated exchange rate time
  • Clean and responsive graphical interface
  • Clear/reset functionality

Technologies Used

  • Python
  • Tkinter (GUI Development)
  • Requests Library
  • JSON Handling
  • PIL (ImageTk for UI elements)

Concepts Applied

  • API integration using HTTP requests
  • Parsing JSON responses
  • Event-driven programming
  • GUI layout design (Frames, Labels, Entry, Buttons)
  • Dynamic data population in dropdown menus
